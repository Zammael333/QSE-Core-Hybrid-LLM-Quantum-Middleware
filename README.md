# QSE-Core: Hybrid LLM-Quantum Middleware

Middleware de alto rendimiento para simulación topológica y gestión SRE.

## Despliegue
1. `pip install -r requirements.txt`
2. `sudo python3 src/oracle_llm_fluid.py`
3. `sudo python3 src/iaq_gatekeeper.py`

## Seguridad
Use `export NEXUS_KILL_SWITCH="TRUE"` para abortar procesos inmediatamente.
